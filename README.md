# 12xue 开源项目
本仓库为 12xue 相关程序的官方开源源代码，**仅用于学习、研究及合规场景下的二次开发**。
main分支为12学客户端源代码，master则是一个可以修改的代码，本代码使用[apktool](https://github.com/iBotPeaches/Apktool)来进行编译和解包。

## 项目说明
- 代码归属：12xue 官方授权开源
- 核心用途：提供 12xue 程序的完整源代码，便于开发者理解系统架构、进行定制开发、漏洞修复及功能拓展
- 适用场景：个人学习、企业内部合规适配、开源社区技术交流


### 部署步骤
1. 克隆仓库到本地
   ```bash
   git clone https://github.com/LafiteLF/12xue-crack.git
   cd 12xue-crack
   ```
2. 使用apktool项目
   ```bash
   java -jar <你的apktool包的名字> b -o <你的自定义名字> # 或对应启动脚本
   ```
