# 12xue-LafiteWuLib
## 本项目为12xue的二次创作与修改提供代码

本项目将[12xue](www.12xue.com)机构的代码开源，并提供修改。
免责声明：此项目严禁为一切侵害设备和应用的程序提供代码，但不限于病毒，钓鱼等。

本项目的两个分支：
main:12xue原版代码，不过修改了一点，具体在AI伴学与主页面处，其他皆为原版
master:经过修改过的12xue代码，且在一些地方补全了注释

构建教程：
使用[apktool](https://github.com/iBotPeaches/Apktool)来进行构建和解包。
构建：
```
java -jar <apktool的包的名字> b <12xue源码文件夹> -o <12xue的apk名字>
```
